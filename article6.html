조선일보에서 제공하는 2018년 전국동시지방선거 특집 사이트 개설 프로젝트로, 예비 후보자 채널 신설과 지역별 뉴스 콘텐츠를 강화하여 선거 관련 정보를 종합적으로 제공하였습니다.

Project Features

예비후보자 채널 신설예비후보자가 자신의 홍보 게시물을 직접 업로드 할 수 있는 예비 후보자 채널 및 관리자 신설

지역별 뉴스 강화사이트맵 설계 과정에서 지역별로 분류한 뉴스 메뉴를 신설, 메인에 실시간 지역별 뉴스를 노출하여 지방선거의 특색을 살리고 로컬 뉴스를 강화

선거 당일 실시간 대응투표현황, 출구조사, 개표현황, 당선현황 등의 당일 선거 정보를 실시간으로 대응하여 유저에게 필요한 정보를 신속하게 제공

인터렉티브 그래프현황 그래프에 인터렉션을 적극적으로 활용하여 많은 정보를 한 눈에 쉽게 확인할 수 있도록 설계

이미지 자동 크롭 기능수많은 지방선거 후보들의 이미지를 자동으로 정렬하여 크롭하는 기능을 추가하여 신속한 대응에 활용

기존 정당과 무관한 컬러 사용여러 정당의 특정 정치색을 피하고 중립적인 이미지를 강조하기 위해 그라데이션이 포함된 컬러 사용
